# Agenci i własne instrukcje (Agents & Custom Instructions)

Rozdział wyjaśnia jak tworzyć i używać agentów (specjalizowanych osobowości AI) oraz jak dodać stałe instrukcje kontekstowe dla sesji Copilot.

## Główne tematy

- Format plików agentów (`.agent.md`)
- Kiedy stosować agenta vs. skill
- Przykłady: python-reviewer, pytest-helper

## Praktyka

- Utwórz prostego agenta, skopiuj szablon z `samples/agents/` i przetestuj go w sesji Copilot.
