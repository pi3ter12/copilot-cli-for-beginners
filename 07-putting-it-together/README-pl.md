# Wszystko razem (Putting It All Together)

Ostatni rozdział scala materiał: łączenie agentów, skills i MCP w pełne przepływy pracy — od pomysłu do scalonego PR w jednej sesji.

## Cele

- Orkiestracja agentów i skills
- Automatyzacja (hooki, CI)
- Przykładowe końcowe przepływy (bug fix, feature, onboarding)

## Ćwiczenie końcowe

Zaimplementuj funkcję "list unread" w aplikacji książkowej korzystając z planowania, agentów i generowania testów.
