# Przepływy pracy deweloperskiej (Development Workflows)

W tym rozdziale znajdziesz wzorce pracy: przeglądy kodu, generowanie testów, debugowanie i refaktoryzację z użyciem Copilot CLI.

## Co obejmuje

- Code review i automatyczne sprawdzenia
- Generowanie testów (pytest)
- Refaktoryzacja i naprawa błędów

## Ćwiczenia praktyczne

- Użyj Copilot, aby wygenerować testy dla modułu `BookCollection`
- Przeprowadź przegląd bezpieczeństwa dla przykładowego pliku
