## GitHub Copilot CLI dla początkujących (skrót)

✨ Naucz się przyspieszać pracę programistyczną dzięki asystentowi AI w terminalu.

Ten kurs pokazuje, jak używać GitHub Copilot CLI, pracując na przykładzie prostej aplikacji Python do zarządzania książkami. Materiały obejmują instalację, tryby pracy, analizę kontekstu, generowanie testów, tworzenie agentów i integrację z MCP.

### Główne punkty

- Struktura kursu: rozdziały 00–07 z ćwiczeniami i przykładami
- Przykłady: główny projekt w `samples/book-app-project/` (Python) oraz wersje JS i C#
- Narzędzia: skrypty w `.github/scripts` do generowania demonstracji; `npm run release` tworzy materiały demo

### Jak zacząć

1. Przejrzyj rozdział 00 – Quick Start, aby zainstalować Copilot CLI
2. Użyj Codespaces lub zainstaluj lokalnie (npm/Homebrew/WinGet)
3. Przejdź do `samples/book-app-project` i uruchom przykładową aplikację

Szczegóły i pełne treści znajdują się w poszczególnych rozdziałach katalogu 00-07.
