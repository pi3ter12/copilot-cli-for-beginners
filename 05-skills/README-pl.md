# Umiejętności (Skills)

Rozdział opisuje skills — moduły, które Copilot ładuje automatycznie, gdy pasują do zapytania. Zawiera szablony i instrukcje tworzenia SKILL.md.

## Najważniejsze punkty

- Struktura skill: folder z plikiem `SKILL.md` zawierającym frontmatter YAML
- Skills uruchamiają się automatycznie na podstawie opisu
- Przykłady: code-checklist, pytest-gen, commit-message

## Jak zacząć

Skopiuj przykładowy katalog `samples/skills` do `~/.copilot/skills/` lub `.github/skills/` i testuj wywołania w Copilot CLI.
