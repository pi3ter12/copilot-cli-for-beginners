# MCP Serwery (MCP Servers)

Ten rozdział pokazuje jak podłączyć Copilot do zewnętrznych źródeł danych przez MCP (Model Context Protocol). Zawiera przykłady serwerów i konfigurację.

## Co omówiono

- Typowe MCP servers (GitHub, filesystem, custom)
- Konfiguracja `~/.copilot/mcp-config.json`
- Przykładowy prosty serwer Python (lookup book)

## Praktyka

- Uruchom przykładowy MCP server i użyj go z Copilot, np. do wyszukiwania informacji o książkach.
