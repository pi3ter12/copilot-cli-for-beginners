# Przykładowe źródła (Sample Source Code)

Te pliki JS/React są opcjonalnym odniesieniem. Główny przykład kursu to aplikacja Python w `samples/book-app-project/`.

## Zawartość

- `api/`, `auth/`, `components/`, `models/`, `services/`, `utils/` — przykładowe pliki front-end/backend
- `refactor-me.js` — plik do ćwiczeń refaktoryzacji w rozdziale 03

Używaj składni `@` w Copilot CLI, by odwoływać się do tych plików w zapytaniach.
