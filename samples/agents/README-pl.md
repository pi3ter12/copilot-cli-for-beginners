# Przykładowi agenci (Sample Agents)

Szablony agentów, które można skopiować do `~/.copilot/agents/` lub `.github/agents/`.

## Zawartość

- `hello-world.agent.md` — prosty agent powitalny
- `python-reviewer.agent.md` — agent recenzujący kod Python
- `pytest-helper.agent.md` — agent pomagający w generowaniu testów

Pliki agentów zawierają frontmatter YAML i instrukcje działania.
