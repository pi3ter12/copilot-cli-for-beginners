# Aplikacja kolekcji książek (Book App)

Prosta aplikacja Python do zarządzania kolekcją książek — dodawanie, usuwanie, listowanie i oznaczanie jako przeczytane.

## Pliki

- `book_app.py` — interfejs CLI
- `books.py` — logika kolekcji książek
- `utils.py` — funkcje pomocnicze
- `data.json` — przykładowe dane
- `tests/` — testy pytest

## Uruchamianie

```bash
python book_app.py list
python book_app.py add
python -m pytest tests/
```
