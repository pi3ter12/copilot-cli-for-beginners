# Przykładowy kod z błędami (Buggy Code)

Zbiór przykładów zawierających celowo wprowadzone błędy bezpieczeństwa i logiczne (JS i Python). Służy do praktyki przeglądów bezpieczeństwa i testowania.

## Zawartość

- `js/` — przykłady Node.js z błędami
- `python/` — przykłady Python z błędami

## Ćwiczenia

- Audyt bezpieczeństwa
- Naprawa krytycznych błędów
- Generowanie testów wykrywających błędy

Używaj tych przykładów w Rozdziale 03 i podczas ćwiczeń.
