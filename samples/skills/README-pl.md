# Przykładowe umiejętności (Sample Skills)

Szablony skills gotowe do użycia. Kopiuj katalogi do `~/.copilot/skills/` lub `.github/skills/` by testować.

## Dostępne przykłady

- hello-world — minimalny przykład
- code-checklist — lista kontrolna jakości kodu (Python)
- pytest-gen — szablon generowania testów
- commit-message — generowanie wiadomości commit zgodnych z Conventional Commits

Instrukcje tworzenia własnych skills i ich struktury znajdują się w oryginalnym README.
