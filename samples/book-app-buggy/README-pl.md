# Aplikacja książkowa — wersja z błędami (Buggy)

Ten katalog zawiera celowo wadliwą wersję aplikacji książkowej do ćwiczeń debugowania w Rozdziale 03. NIE naprawiaj tych błędów — są one przeznaczone do nauki.

## Cel

- Ćwiczenia znajdowania i naprawy błędów przy użyciu Copilot CLI

## Przykładowe problemy

- Wyszukiwanie rozróżniające wielkość liter
- Wycieki plików (brak context manager)
- Brak walidacji roku

Używaj Copilot, aby znaleźć i naprawić błędy podczas ćwiczeń.
